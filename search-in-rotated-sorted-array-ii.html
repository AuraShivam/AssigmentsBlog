Search in Rotated Sorted Array II

Difficulty: Medium

Problem Summary

Given a rotated sorted array with duplicates, determine if a target exists using a modified binary search.

Approach

Use binary search with duplicate skipping to decide which half is sorted, reducing search space.

C++ Solution


class Solution {
public:
    bool search(vector<int>& nums, int target) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == target) return true;
            
            if (nums[left] == nums[mid] && nums[mid] == nums[right]) {
                ++left;
                --right;
            } else if (nums[left] <= nums[mid]) {
                if (nums[left] <= target && target < nums[mid]) right = mid - 1;
                else left = mid + 1;
            } else {
                if (nums[mid] < target && target <= nums[right]) left = mid + 1;
                else right = mid - 1;
            }
        }
        
        return false;
    }
};
