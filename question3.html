Given a positive integer num, return true if num is a perfect square, false otherwise.
A perfect square is the product of an integer with itself.
You must not use any built-in library function, such as sqrt. 

Example 1:
Input: num = 16
Output: true
Explanation: 4 * 4 = 16 and 4 is an integer.

Example 2:
Input: num = 14
Output: false
Explanation: 3.742 * 3.742 = 14 and 3.742 is not an integer.

Constraints:
1 <= num <= 231 - 1

ğŸ’¡ Intuition

Use binary search to check for an integer whose square equals num.

ğŸ” Approach

  1. Initialize search space as [1, num]
  2. For each integer mid in the range, compute mid * mid
  3. If mid * mid == num, return true
  4. If mid * mid < num, search right half; else search left half
Time Complexity: O(log n)
Space Complexity: O(1)
ğŸ’» C++ Solution
class Solution {
public:
    bool isPerfectSquare(int num) {
        long left = 1, right = num;
        while (left <= right) {
            long mid = left + (right - left) / 2;
            long sq = mid * mid;
            if (sq == num) return true;
            else if (sq < num) left = mid + 1;
            else right = mid - 1;
        }
        return false;
    }
};
