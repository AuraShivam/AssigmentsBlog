You are given an m x n integer matrix with the following properties:
- Each row is sorted in non-decreasing order.
- The first integer of each row is greater than the last integer of the previous row.
Given target, return true if target is in matrix or false otherwise.
Write solution in O(log(m * n)). 

Example 1:
Input: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
Output: true

Example 2:
Input: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13
Output: false

Constraints:
m == matrix.length
n == matrix[i].length
1 <= m, n <= 100
-104 <= matrix[i][j], target <= 104

ğŸ’¡ Intuition

Treat the matrix as a virtual 1D array and apply binary search.

ğŸ” Approach

  1. Map the 2D matrix to 1D: index k maps to row k / n and column k % n
  2. Apply binary search between 0 and m * n - 1
  3. Check matrix[mid / n][mid % n] against target
Time Complexity: O(log(m * n))
Space Complexity: O(1)
ğŸ’» C++ Solution
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = matrix.size();
        int n = matrix[0].size();
        int left = 0, right = m * n - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            int mid_val = matrix[mid / n][mid % n];
            if (mid_val == target) return true;
            else if (mid_val < target) left = mid + 1;
            else right = mid - 1;
        }
        return false;
    }
};
